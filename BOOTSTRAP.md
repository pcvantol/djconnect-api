# DJConnect API Repository Bootstrap

This repository owns the Cloudflare Worker/D1 trust and relay boundary. It
adopts AI-Native Engineering Operating System 2.2 from
`pcvantol/djconnect/docs/governance/PLATFORM_ARCHITECT_SYSTEM_INSTRUCTIONS.md`
by reference, never by copying central governance.

Start with `git switch main`, `git pull --ff-only`, current-main/clean-tree and
predecessor verification. Read local records and perform an implementation-
reality check; reconcile `MERGED_UNRECONCILED` before work. Lifecycle:
`LOCAL_IN_PROGRESS`, `REVIEWABLE_FROZEN`, `MERGED_UNRECONCILED`,
`MERGED_RECONCILED`; branch cleanup is fail-closed.
