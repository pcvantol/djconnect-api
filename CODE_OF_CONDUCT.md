# DJConnect Code of Conduct

## Our Pledge

DJConnect is a community project for people who care about music, Home Assistant,
local-first devices and useful software. We want this repository to be
welcoming, practical and safe for contributors with different backgrounds,
experience levels and perspectives.

We pledge to make participation in this project a harassment-free experience for
everyone, regardless of age, body size, visible or invisible disability,
ethnicity, sex characteristics, gender identity and expression, level of
experience, education, socio-economic status, nationality, personal appearance,
race, caste, color, religion, sexual identity and orientation, or other similar
personal characteristics.

## Expected Behavior

- Use welcoming and inclusive language.
- Be respectful of differing viewpoints and experiences.
- Give and receive constructive feedback with care.
- Focus criticism on ideas, code and project outcomes, not people.
- Respect privacy and avoid sharing secrets, tokens, personal data or private
  communications without permission.
- Be patient with new contributors and direct them to the relevant docs when
  possible.

## Unacceptable Behavior

- Harassment, intimidation, stalking or sustained disruption.
- Insults, personal attacks or demeaning comments.
- Sexualized language or imagery, and unwelcome sexual attention or advances.
- Publishing someone else's private information without explicit permission.
- Deliberately posting secrets, credentials, tokens or private artifacts.
- Other conduct that would reasonably be considered inappropriate in a
  professional open-source community.

## Scope

This Code of Conduct applies in all DJConnect project spaces, including GitHub
issues, pull requests, discussions, documentation, release notes and any
community support spaces that clearly represent the DJConnect project.

## Reporting

Report concerns to the project maintainer:

- Peter van Tol, via GitHub: `@pcvantol`

Please include as much relevant context as you safely can, such as links,
screenshots, dates, involved usernames and what outcome you are hoping for.
Reports will be handled with care and discretion.

## Enforcement

Maintainers may remove, edit or reject comments, commits, code, issues, pull
requests or other contributions that do not align with this Code of Conduct.
Enforcement will aim to be fair, proportionate and focused on protecting the
community.
